# Heart Disease Prediction — ML Project

A Machine Learning classification project that predicts 
the probability of heart disease in patients using clinical data.

## Problem Statement
Early detection of heart disease can save lives. This project builds 
an ML model to classify patients as high or low risk based on 
clinical features.

## Dataset
- Source: Heart Disease Dataset (CSV)
- Features: 13 clinical features including age, cholesterol, 
  blood pressure, chest pain type
- Target: Heart disease present (1) or absent (0)

## Models Used
| Model | Accuracy |
|---|---|
| Random Forest Classifier | 80% |

## Tech Stack
Python, Scikit-learn, Pandas, NumPy, Matplotlib, Seaborn

## Key Steps
- Exploratory Data Analysis (EDA)
- Outlier detection and treatment
- Feature correlation analysis
- Random Forest model with GridSearchCV tuning
- Model saved using joblib

## How To Run
1. Clone the repository
2. Install: pip install -r requirements.txt
3. Open HeartDiseasePrediction.ipynb in Jupyter or Colab
4. Run all cells
